JavaScript Language

Parentheses ( )  in JavaScript are used :

  1. to make function calls - e.g function myFunc( )
  2. to surround conditional statements - e.g if (condition1)
  3. for grouping to determine Order of Operations - e.g if ( (4 + 5) * 2)

Braces { }  in Javascript are used :

  1. during the declaration of Object Literals - e.g var drink = { manyDrinks: {a: "juice", "b": "milk"}, 7: "soda" };
  2. to enclose blocks of code (function definitions, conditional blocks, loops, etc).

Brackets [ ]  in Javascript are typically used for accessing the properties of an Object (or the elements of an Array) - e.g var mylist = [1,2,3,4];

In Javascript, there is no meaningful difference between the single quotes ‘ ‘ and double quotes “ ”. In practice, single quotes are just more convenient. However, there are some exceptions to this. For example, when formatting JSON data, valid JSON only takes the double quotes. The choice to use either single or double in the rest of a Javascript project is personal, though it is best practise to keep it consistent to avoid hard to find errors.
